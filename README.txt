AfriBooms Afrikaans Universal Dependencies Treebank
===================================================
This treebank is based on the AfriBooms Dependency Treebank (TDT), which is available from the South African Language Resource Management Agency, Potchefstroom.

Size of the differents parts
  - train size =  1546 sentences
  - dev size   =  194 sentences
  - test size  =  194 sentences

We will be working on manually verifying the treebank.
